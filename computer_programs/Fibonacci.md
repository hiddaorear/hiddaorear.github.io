## 计算 Fibonacci numbers

### 定义

$$F_0 = 0$$
$$F_1 = 1$$
$$F_n = $$F_n-1 + $$F_n-2$$

### 依据定义实现

``` C++

int fib0(int n) {
	if (n == 0) return 0;
	if (n == 1) return 1;
	return fib0(n - 1) + fib0(n - 2);
}

```
