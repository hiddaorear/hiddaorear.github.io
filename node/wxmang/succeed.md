# 成功

#### 文小芒先生

其实我们所有人都是事后诸葛亮，在当是时，身在庐山，都是晕乎乎的，除非是毛主席这种天纵英才。所以才有这么多经验总结和后悔药。

**我们无法控制环境，我们只能把握自己（实际90％的人都把握不住自己的贪、急、忿）。其实只要能够使自己的时间、精力、体力实现最优配置，如果正好乘上大势，就必然是大成功。所以先做好自己的事情：尽量使自己的时间，精力和体力利用率最高投入产出最优，不浪费，不做无效劳动，不做重复劳动，就已经离成功不远了。其他不可预测，也不需要预测，大势来了，顺势而为即可**。

别人的经历都是无法复制的，甚至都没有参考价值。例如我的，80年代初国家严重缺乏专业人才，那时但凡有点知识，口齿清楚，就能得到很好的位置和重用。所以有人曾经怀疑我不到30岁多一点就做总经济师，然后又做总裁等等。其实就像我们现在无法理解林彪20多岁做军团长一样。是时代，是命运。

成功都是自己努力积累知识经验，同时正好抓住机会的结果。前者需要勤奋和聪明，后者需要敏感和运气。至于真的有了机会，想把位置坐稳，我想就需要坚韧不拔，决不后退的精神，以及随时准备在关键时候豁出去，孤注一掷的决心。其他没有什么诀窍。你有上述素质，人品又是诚恳和踏实的，那么就会有人赏识你，启用你，给你机会。

不要以为人生是买彩票，没这回事情，人生是没有什么意外的。

就我观察（很多人的成长过程）：

首先的人品好，诚实，稳重，不然没有成功的机会，那种口若悬河，信口胡吹的没有机会，因为一旦被人看穿会死得很惨，而且一定会被人看穿；

其次是心理素质好，坚韧不拔，稳定深沉，吃苦耐劳。投机取巧的人，过于敏感的人，患得患失的人都很难成功；

三是有理想，有目标，有上进心，那种终日无所事事，游手好闲，好吃懒做的不会成功；

第四是受过很好教育，爱学习，不断进步，那种不读书不看报的混混没机会；第五是家庭负担不重，不然会急功近利，病急乱投医。。。。。。我认为以上可称为潜力股。先写这些，其他等想起再补充。

我想失败的主要原因：

一个是不自信，这是最常见的，不是低估自己，就是提虚劲；

二是不够坚韧不拔，随时准备逃跑放弃；

三是不够勇敢，面对危机或机会，犹豫不决，斤斤计较，贻误战机；

四是不够乐观，总是看到失败的可能，总是在泄自己的气，也在毒化群体意志，瓦解群体斗志；

五是不够自尊，经常把自己当成店小二，尤其面对强势对手或者强势合作者，卑躬屈膝，被人鄙视。

你这夸张了，我基本从不预测什么吉凶，我只鼓励自强不息，自信，自尊，勇敢，诚实，坚韧。

这就是大多数人不成功，少部分人成功的秘密：他们一直能够保持激情，坚持不懈。

就是你说的心气。一个三五年就消失心气的人，还能指望以后百折不挠，坚韧不拔？

**你的问题就是不够坚韧不拔，容易这山看到那山高，所以总是踏空。如果你不能踏实做事情，这辈子将一事无成**。

就算卖油条，也要做熟做透，做到最好。没有人会随随便便成功。每个成功的人背后都是艰辛的努力，甚至红色贵族也如此，你知道586付出什么常人不能忍受的努力吗？

普通人的实用主义就是：兢兢业业，尽职尽责做好本职工作，站稳脚跟，解决生存；同时骑驴找马，有好的机会（看个人判断：更有发展前途，更多钱，更多权等等），绝不犹豫，绝对不放过，随时准备机会降临。至于浪费时间的无用功（与自己目标不相干的事情都是无用功）尽量不做，盯住目标，不放弃，坚韧不拔，坚持不懈，任何时候都不悲观，不泄气，斗志昂扬。

以前人家说，长什么样子，30岁以前怪父母，30岁以后怪自己；其实，放在生活上也一样，30岁以前，过得怎么样，看遗传，环境，教育，30岁以后，看自己是不是努力，是不是坚韧不拔。

跟我同龄的人中间，愤世嫉俗，怀才不遇，怨天尤人的不在少数，他们中间毕业于北大清华科大的也不在少数，总之他们对自己绝望，对别人嫉恨。

究其原因，我想他们不够努力，或者说不够踏实有关。有的家伙在20多岁时以投机取巧而沾沾自喜，有的以能够拉大旗作虎皮而不可一世，可是没有好父母和好丈人的人，在这个社会还是靠竞争力。一个人不能一辈子靠玩虚的过日子，一旦真刀真枪，不踏实的人，就是人家的垫脚石。

有时乐观能够拯救一船人。

比如以前我说过的一个例子，刚接管一个长期停产的企业，恢复生产不到一个月，企业主要设备就损坏，一旦半成品耗尽，就要全面停产，为了稳住人心（当时还是人心惶惶，还没搞定），不至于带来混乱，我只能白天如常继续安排上班，晚上去现场悄悄指挥抢修。其实我当然知道停产的后果，搞不好就是通钢动乱的下场，我虽然不至于被工人打，但是接管彻底失败是肯定的。

结果熬过来了。我想靠的就是我的坚韧和乐观。我不相信天要绝我。我更相信天无绝人之路，更相信事在人为。

要说对企业熟悉程度和老谋深算，我不如原来企业的那些老家伙，但是他们后来非常服气我，可以说鼎力支持。因为在所有人都唉声叹气，大骂老天不公时，我没有选择撤退逃跑回北京（本来他们猜想我这个小书生一定会害怕逃跑的），而是与他们同舟共济。

**人活的就是一口气**。

**我不太理解这个概念，因为如果说善于搞好人际关系就叫有情商，显然不对，因为失败者中这种滑头多得很。我觉得成功主要还是以诚待人，勤恳做事，坚韧不拔**。

这个真把我难住了，因为跟你跟的领导个人特点有关，有的领导喜欢部下是韩信类型的，能够解决实际问题，冲锋陷阵，有的喜欢张良类型的，能够足智多谋，运筹帷幄，有的喜欢萧何类型的，能够安定后方，处理烂摊子。

这个一定投其所好，看你的领导需要什么类型的人。不过一般规律是（排名有先后）：

忠诚可靠，情绪稳定，坚韧不拔，理性宽容，成熟稳重，头脑清楚，表达严谨，吃苦耐劳，胆大心细等等，当然最好能够写得好文章，下得基层搞定麻烦，能说会道，还要把功劳给领导。 如果做到这些，可以期待以后你就是单位一把手了。

没本事人才当公公，献媚谄笑，或者行贿受贿。

就我的经历来看，面对困难悲观失望的人，甚至丢盔弃甲的人；面对危机灰心丧气的人，甚至逃跑躲避的人，很难有机会成功，更不用说建功立业。

毛主席早就说过：我们的同志，在困难的时候，要看到成绩，要看到光明，要提高我们的勇气。毛主席又说：胜利往往在最后的坚持一下之中。

很多人没有坚持下来，非常可惜，其实经常离胜利就差一步。

乐观，不是自我催眠，是一种生活态度。

**整体来看，谋事在人，成事在天，但是在局部情况下，事在人为。人与人成就不同，路径不同，其实就取决于几个关键点的事在人为不同。在关键节点上，自信，勤奋，勇气，果断，可以彻底把一个人与芸芸众生区分出来**。

**不要着急，耐心积累能力、经验和知识，等待机会，总是会有的。如果不知道干什么，就读自己喜欢的专业的书。（不是小说）少上网，上网学习效率太低，浪费时间。一旦有机会，不要犹豫不决，敢于豁得出去**。

你真的知道自己喜欢什么吗？愿意为自己喜欢的东西贡献一生精力和时间吗？如果不知道，先把这个问题搞清楚。

我想一句话可以说清楚这个要求：适可而止，或者说量力而行，循序渐进。有进取心是必须的，就像人必须吃饭，而且饭要一口口吃，目标也要一步步确定，一步步实现。但是要知道自己的饭量，不能无限吃下去，把自己撑死。所以要自强不息，勇猛进取，同时也要有自知之明，不能贪得无厌。

这大概就是中国人的中庸之道吧。

谢谢你的夸奖，我从小就不经夸，我母亲说一夸就人来疯。以后我得更努力的来贡献自己的失败教训了（我认为成功基本不可复制，而失败却可以启发人，提醒人，所以失败教训比成功经验重要）。我想别人怎么做我们不要非议，因为各人有各人的想法和处境，这是他们的自由。我们只能做好自己的事情。

举例来说，就我个人而言，对10多年前做事情急功近利，投机取巧，半途而废，教条主义，理想主义，好大喜功等等和为人处事的霸道，不合作，不宽容带来的失败或后果一直耿耿于怀，经过多年反思和刻骨铭心的自我解剖，才有勇气向别人坦承，并希望别人接受教训不要重蹈覆辙。这个不是每人都能做得到，每人都有义务做的。所以不能苛求别人。

**只要做到不自欺，不欺人，问心无愧就足矣。至于事功成就，天时地利人和一样不能少。个人能力和具有的技术只是必要条件之一，人的心态最重要：不贪，不急，不忿。因为有明确的目标，所以可以做到不贪，有所为有所不为；因为有完整的计划，所以可以做到不急，按部就班；因为有自知之明，知己知彼，所以可以做到不忿，不嫉妒不眼红**。

这是我对成功的观察。

其实我经常在想的问题与你类似，那就是什么叫成功。我们一生除了追求自己内心的平静、满足、问心无愧外，可能就是追求自己智力、精力和体力的最大效用，而衡量的标准就是成功。

成功既然是个价值判断指标，就需要有评价体系，不同的价值体系，显然成功标准是不同的。

例如我有一个同学现在是第一流的外科医生，救人无数，有时我就很羡慕，觉得当年我去学医，说不定也能救人性命。我觉得他巨成功，与他相比，我太不成功。相反，他觉得我很成功：体验了人生的酸甜苦辣，波澜起伏，世态炎凉，生命充满传奇和惊喜。而他30年来就在病房和手术台转悠，平淡无奇。

产生这种评价差异的原因是我们的价值体系不同，我实用，看重事功；他超脱，看重个人体验。但是就如不能人人都去做外科大夫一样，还得有人去做蓝领工头，不然外科大夫连做手术的医院都不会有。这个社会是基于分工存在的。所以只要做好自己本分，也就能心安了。

成功本来是个体的体验，但是需要社会的认同。如果你自信，你就是成功的。如果你不自信，你就需要别人给你打分。总的来说，自我满足最重要。

这个只能因人而异，不同人选择不同，没有公式，简单来说，至少包括以下因素：个人兴趣，个人性格，个人神经类型，个人专业，个人专长，人生目标，背景（父母或岳父母的人脉），地域，等等。 做职业规划，不能考虑必要性，而要优先考虑可行性。个人没竞争力的职业，选择也只是别人垫脚石而已。

不能选赚钱最多，最热门的行业，因为别人也会选，除非你有金刚钻，例如有奇才，有特殊技艺，或者有好父母或岳父母。 选次一等的行业，比较有发展机会。就像我的亲身体会，78年时，就算是省级高考状元，如果一般智力，最好不进科大数学系，理论物理系之类，因为你无法与天才们同场竞技，只能当分母，但如果去了复旦，上海交大，南开这类学校，没准能够成为大腕。扬名立万。

过奖了，我其实是一招鲜吃遍天。我没有管理过重型机械企业，对它们的研发、生产和营销都不熟悉，我唯一能够理解的是它们的财务问题和管理流程问题。我的看家本领其实一个是全面预算管理和成本管理（还主要专注于物流成本控制），一个是管理信息系统里面的资源最优配置模型，专业化分工网络模型和实时监控模型。

其实我写这些帖子，目的只有一个，那就是展示只会很少的技术，如何理解复杂的问题，也即一招鲜如何使用得当，变成吃遍天。人的精力和智力都有限，不可能什么都学到极致，但是如果能够把基本的东西融会贯通，就能做事情。

我刚进系统所时，有一次在数学所图书馆看武侠小说，吴文俊先生去图书馆找夫人（他夫人是数学所图书馆馆长），看到我在看闲书，问：你的专业是什么，我说：数理统计，他说什么课学得最好？我说：线性代数（当年科大线性代数课程号称亚洲第一难），他说：不错了，会线性代数，就能做很多工作了，刘书麟（这是系统所一个做数学物理的研究员，科大数学系59级校友，吴先生的学生）就靠线性代数就作出很重要的工作。**不必一味追求工具先进，能把工具用熟了比什么都强**。

所以我觉得贪多嚼不烂，不需要什么知识和工具都学习，不如熟练应用自己最熟悉的知识和工具，玩到极致（其实吴文俊先生就是把一些简单工具玩到极致，玩到山花烂漫的人），否则就是口水党。再说一个人一生都在学习做事情的工具，什么时候才能做事情，不能一辈子都是学生吧。

那当然，一般情况下，家长就是孩子的榜样和模板，家长为人处世能力和社会道德水平的上限，就是孩子的上限。

中国这个强调团体意识和集体主义的文化氛围下，为人处世能力和道德水平，基本决定了孩子能够走多远。就算天纵英才，除非搞纯数学和理论物理这种可以与世隔绝的学问，否则只要跟人打交道，是否能够成功，人生是否顺利，与人和谐相处能力的作用大于智力和体力，更大于知识储备。

**一个人如果像你这样，就是个软弱可欺的对象，根本没机会成功。做事情必须霹雳手段，菩萨心肠。没有前者，做不成事情，没有后者，凝聚不住人。我的原则是打击报复不过夜，感恩戴德不隔天。帮别人都是在帮自己，哪有人是雷锋。独善其身的人，只能耍单帮，单打独斗**。太看重自己的财产了，做事情是舍不得孩子套不到狼，不敢豁出去的人，什么都不会有的。等你走上社会，你就知道，成功的人都会在某一时候孤注一掷，并不是他们知道必然会成功，而是知道不这样做会终生后悔。书生最大的问题是关键时候豁不出去，瞻前顾后。你什么时候不这么胆怯了，你就成熟了。哪位老兄当时不这么干，他就彻底完蛋，再无机会。因为再无人来追随：没钱没权的人想成大事，只能靠人心了。
